## Risk Analysis

 This is a risk analysis plan with a scheduled strcutured for this website using the TAME method and a Work Breakdown Structure.

#### Transfer 

Risk: Images for inventory of items may not meet quality standards
 
 Strategy: Construct professional like photos of the inventory to establish a good appeal for each item. This well help keep the website in high quality and trusted. 
 
 Outcome: This will take some time spent to produce orginal media but ensures the quality is consistent. 

 #### Accept

 Risk: Minor layout inconsistiencies may occur

 Strategy: Small visuals will be distinct across different browsers and devices. These issues will be addressed only if they affect users significantly.

 Outcome: Prevents any reowkring that is unecesscary

 #### Mitigate

 Risk: Missing required elements

 Strategy: Use the checklist provided before submission and validate all files. A brief review will done before finalizing this website.

 Outcome: ensures project stays within the scope and required guidelines.

 #### Eliminate
 Risk: overcomplications to the site with payment processing or backended systems.
 
 Strategy: Exclude these overcomplications and dynamic features from this project at this time.
 
 Outcome: Removes complexities and keeps the project as a simple website for now.
